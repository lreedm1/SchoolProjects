Recursive

  1. Compare the middle element of the array with the target value.
  2. If the middle element is equal to the target value, return the index of the middle element.
  3. If the middle element is greater than the target value, recursively search the left half of the array.
  4. If the middle element is less than the target value, recursively search the right half of the array.
  5. If the target value is not found in the array, return -1.

Iterative

Here are the steps to perform an iterative binary search:

  1. Initialize low and high variables to the start and end of the array, respectively.
  2. Repeat the following steps until low is greater than high:
    1. Compute the middle index as the average of low and high.
    2. Compare the middle element of the array with the target value.
    3. If the middle element is equal to the target value, return the index of the middle element.
    4. If the middle element is greater than the target value, update the high index to be the index before the middle index.
    5. If the middle element is less than the target value, update the low index to be the index after the middle index.
  3. If the target value is not found in the array, return -1.

For more information on binary search, see Wikipedia's page on binary search.
